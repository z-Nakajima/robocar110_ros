# RoboCar 1/10 ROS nodes for teleoperation #
***

## Dependencies ##
***

- ROS environment
- joystick ROS drivers
```bash
    sudo apt-get install ros-${ROS_DISTRO}-joy
    sudo apt-get install ros-${ROS_DISTRO}-joystick-drivers
```
